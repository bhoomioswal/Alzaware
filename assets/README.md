# Assets

This folder contains the visual assets for the AlzAware prototype.

## Contents

### Screenshots
The `screenshots/` folder contains images of different modules of the application, including:

- Home Page
- User Registration
- MRI Prediction
- Facial Emotion Recognition
- Speech Emotion Recognition
- Memory Assessment
- Cognitive Assessment
- Final Dashboard

### Demonstration Video
The `demo/` folder contains a demonstration video showcasing the complete workflow of the AlzAware prototype, including user registration, assessments, AI model predictions, and the final dashboard.

These assets are provided to help users understand the functionality and user interface of the project.
